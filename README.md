# 🌐 My Portfolio

This is my personal portfolio website built with React and Vite. It showcases my skills, projects, and contact details.

---

## 🚀 Tech Stack

* ⚛️ React
* ⚡ Vite
* 🎨 CSS / Tailwind

---

## 📂 Features

* Responsive design
* Project showcase
* About section
* Contact section

---

## 🛠️ Run Locally

```bash
git clone https://github.com/your-username/my-portfolio.git
cd my-portfolio
npm install
npm run dev
```
